# objektinis-programavimas
This project aims to analyze the performance of a program handling student data through research studies.

### Research Studies

1. **File Generation Study:**  
   Investigates the time required to generate files containing student data. Various data sizes are considered, ranging from 1,000 to 10,000,000 students, with each student having 15 grades.

2. **Program Execution Time Study:**  
   Focuses on measuring the execution times of different parts of the program. It analyzes the time taken for data reading from files, sorting of students into two groups (containers), outputting student groups into new files, and the overall duration of the program. This study utilizes the files generated during the first study.
